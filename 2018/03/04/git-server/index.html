自己搭建git server

在自己的云主机搭建git服务器

环境：
local: Windows10, git bash
remote: centOS7

这里是笔记迁移，大约是在半年前搭建的git server,故没有图示。

一、服务器端创建git用户

  1. 安装git

  2. 创建git用户并且赋予权限

    会在/home目录下看到一个git用户目录

  3. 给git用户分配一个密码

    这个密码用在你后面提交代码的时候使用(然而并没有用到。。。)

二、本机创建ssh key

git-bash中创建一个新的 ssh key
参考GitHub ssh密钥问题

三、服务器上将ssh public key 添加到服务器

  1. cd /home/git/
    git表示用户git
  2. mkdir .ssh
    没有该目录就创建

  3. 进入.ssh目录

  4. 新建文件authorized_keys

  5. 将public key串填入
    如果有多个，则换行追加
    保存退出

  6. 新建文件夹testgit测试
    在其它目录，比如网站根目录,我的是 /usr/share/nginx/html/,新建文件夹testgit测试

五、服务器端创建仓库(repository)测试

  1. 新建git仓库

    新建git仓库，一定要是空仓库，–bare参数不能省略

  2. 修改testgit目录权限

    修改testgit目录权限
    也可以使用chown…自由发挥吧
    如果不修改，那么push的时候会有写入错误

六、本机上使用仓库

  1. 创建空仓库

    在一个空目录里面创建一个空仓库

  2. 添加远程仓库

    这时可以用git_cloud替代211.159.184.137
